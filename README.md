## 실행 방법

main.py 파일이 있는 디렉토리에 upbit.key 파일을 생성한 후 access, secret을 각 줄에 넣어줍니다. 

```
vyGK4x7QAi7UjHxCOV35
FMc4n2H9oOSO2gE1CwEs
```

pyupbit 모듈을 설치합니다. 

```
$ pip install -U pyupbit
```

실행합니다. 

```
$ python main.py
```

프로그램이 시작되면 무한매수 시작 버튼을 눌러 프로그램을 시작합니다.
